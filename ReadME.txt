--------------------------------------------------------------------------------------------------------------------------------------
THis script works with the latest iLearnNYC and apex quiz system.
The purpose is to extract a displayed question and return a correct answer.
Currently the question is opening in a new tab showing googled resulted, development in creating an auto answer system is coming along.
---------------------------------------------------------------------------------------------------------------------------------------
Steps to use this script:

1. Download the chrome extension TamperMonkey
   https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en
   
2. Head over to github and search for "Apex Question Extracter" open this directory
   https://github.com/search?q=apex+question+extracter
   
3. Open the Main Javascipt file "Strip Question.user.js" 
   https://github.com/PwnGoat/Apex-Question-Extracter/blob/master/Strip%20Qustion.user.js
   
4. Copy the entire text inside the file. 

5. Click onthe TamperMonkey Extension and press create new script.

6. Paste in the contents that you've copied.

7. Go take apex quiz :)
